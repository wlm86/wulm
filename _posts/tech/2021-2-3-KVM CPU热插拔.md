#   KVM CPU热插拔

### 1.**virsh 热插拔vcpu**

1. 只能热插vcpu 不能热拔
   例如动态减少vcpu 24->16,会报错。
   ```
   [root@host10573617 ~]# virsh setvcpus 8 16
   error: unsupported configuration: failed to find appropriate hotpluggable vcpus to reach the desired target vcpu count
   ```
